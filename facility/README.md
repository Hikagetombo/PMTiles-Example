# 千葉県の公共施設
[国土地理院の公共施設データ](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-P02-v3_1.html)よりダウンロードしたデータをマーカ表示する。
### DEMO
[マーカ表示](https://hikagetombo.github.io/PMTiles-Example/facility/facility.html)

## このプログラムについて
このプログラムの初期バージョンは自分で作成したもので、千葉県内の公共施設データ（PMTilesベクトルタイル）を読み込み、それを単一色のマーカで表示するものでした。
その後、AntiGravityに依頼して、以下の機能を追加しました。
- 大分類コードに従いマーカを色分けすること
- マーカの凡例を表示すること
- マーカをクリックしたときに、マーカの表示/非表示をトグルすること

途中２回ほど機能確認を行い、問題点を改善してもらいましたがほぼ完璧なプログラムに仕上がりました。
