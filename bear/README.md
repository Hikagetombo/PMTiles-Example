# 東京都内の熊目撃情報
## マーカ表示
東京都内での熊目撃情報を確度別にマーカを色分けして表示する。
- 赤色：高
- 黄色：中
- 青色：低
### DEMO
[マーカ表示](https://hikagetombo.github.io/PMTiles-Example/bear/bear.html)

## アイコン表示
上で表示したマーカをアイコンで表示するよう変更したもの。  
もっと鮮やかなアイコンが欲しかったが入手できず、地味な色合いになってしまった。

[アイコン表示](https://hikagetombo.github.io/PMTiles-Example/bear/bear-image.html)
